No. The scan is read-only. Cleanup uses safe trash first, and each file includes usage context before action.

Yes. Trashed files can be restored quickly during the retention period.

Does it work with WooCommerce?

Yes. Product images and gallery media are included in scanning and usage checks.

No. Scans run in background routines designed to avoid front-end performance impact.
