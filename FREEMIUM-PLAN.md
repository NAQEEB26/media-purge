# WP Media Purge — Freemium Plan (80% Free / 20% Paid)

**Philosophy:** Focus on quality and solving the real problem first. Build trust and audience before monetization. Avoid the "charge for everything" approach that frustrates users.

**Author:** Naqeeb Ul Rehman  
**Website:** https://naqeebulrehman.com

---

## Free Tier (80%) — The Core Value

These features solve the real problem and are **100% free forever**:

| Feature | Description | Why Free |
|---------|-------------|----------|
| **Unused Media Scanner** | Full scan: post_content, post_meta, wp_options, widgets, theme_mods, WooCommerce gallery | Core value — users must trust the scanner first |
| **Page Builder Scanning** | Elementor, Divi, WPBakery, Beaver Builder | Competitors charge for this; it causes the most false positives. Free = trust |
| **"Used In" Preview** | Show exactly where each image is used before trashing | #1 differentiator — never hide this behind paywall |
| **Safe Trash Flow** | Review → confirm → trash with 30-day recovery | Safety is non-negotiable — free |
| **Restore** | One-click restore within 30-day window | Part of safe flow — free |
| **Storage Dashboard** | Total storage, breakdown by type, largest files, potential savings | Helps users see value — free |
| **Bulk Operations** | Select all, filter (type/size/date), bulk trash, bulk whitelist | Essential for usability — free |
| **Export CSV** | Export unused list | Useful for auditing — free |
| **Whitelist** | Protect files from future scans | Prevents accidents — free |
| **Unlimited Scans** | No monthly scan limit | No artificial caps — free |
| **Unlimited Trash** | No monthly trash limit | Users need to clean in bulk — free |

## What We Do NOT Charge For

- Scanning accuracy or scan locations
- "Used in" preview
- Trash or restore
- Storage stats
- Monthly limits (no artificial caps)
- Page builder compatibility
